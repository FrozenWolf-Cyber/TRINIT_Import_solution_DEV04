About EmailedIn


This web application is a submission to the "DEV04" problem statement by the team "import_solution", for the Tri-NITT Hackathon 2022. The main aim of this web app, is to figure out a person's LinkedIn profile using only a Email ID.


The app uses the SignalHire API, to find out the URL to their LinkedIn profile. The API also provides extra information such as name, location, education etc., mentioned in their profile

This is the Github repository of this project.
